# Yoga back
## Requirements
Software 
- java 8
- MySQL
- maven
  
Test Library

- JUnit
- Jacoco

## Setup
> Launch a MySQL Server.

> Execute "ressources/sql/script.sql" .

> Edit the file "back/src/main/resources/application.properties" with your database credentials


## Testing
For launch and generate the jacoco code coverage:
> mvn clean test
